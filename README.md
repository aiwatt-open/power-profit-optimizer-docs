## Overview

- **Base path**: Docs use `/v1/xxx`; actual API is `/api/v1/xxx`
- **Pagination**: List endpoints use `page` and `page_size`
- **Time**: Timestamps are Unix seconds; `created_at`/`updated_at` are ISO 8601
- **Errors**: JSON with `code` and `message`

See the [English README](en/README.md) or [中文 README](cn/README.md) for details.
